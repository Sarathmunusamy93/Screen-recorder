# Screen-recorder
This chrome extension will record the screen in user desired video format. User can save the video with given name and format.

The extension short will display in the popup tray in upper right corner of Chrome browser
 <img width="1500" height="701" alt="image" src="https://github.com/user-attachments/assets/20e29725-950a-47ee-9e2d-18c29a2641d6" />

If user clicks the extension name below screen will display where user can start recording with desire format by select the video format in dropdown.

 <img width="1500" height="678" alt="image" src="https://github.com/user-attachments/assets/b35604ac-1ac2-46c2-9b86-68c4814fc5e9" />

Chrome browser screen popup will be displayed so, that user can select the screen which they want to record. 

  <img width="1500" height="1020" alt="image" src="https://github.com/user-attachments/assets/f4a3c0e3-dd94-466e-937b-266c4d98da82" />

Once user completed with recording, "Stop Recording" button can use to stop recording and they can type the file name and click download to dowload recorded file in local.
